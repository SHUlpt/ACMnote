## 常用方法

|         方法         |                   作用                    |
| :------------------: | :---------------------------------------: |
|      `l.size()`      |            返回链表中元素数目             |
|     `l.empty()`      |                 链表判空                  |
|     `l.front()`      |    引用返回第一个元素，不检查是否存在     |
|      `l.back()`      |   引用返回最后一个元素，不检查是否存在    |
| `l.push_back(elem)`  |               尾部添加元素                |
|    `l.pop_back()`    |             移除最后一个元素              |
| `l.push_front(elem)` |               头部添加元素                |
|   `l.pop_front()`    |              移除第一个元素               |
|   `l.remove(val)`    |           删除所有值为val的元素           |
|  `l.remove_if(op)`   |    删除所有使得`op(elem)==true`的元素     |
|    `l.erase(pos)`    | 删除迭代器pos所指的元素，返回下一元素位置 |
|     `l.clear()`      |               删除所有元素                |
